## Learning Path

### Phase 1: HTML Fundamentals (Beginner)

Start with the basic HTML examples to understand the structure and semantic markup:

1. **[HTML Structure](examples/basic/html-structure.html)**
   - Document structure (`<!DOCTYPE>`, `<html>`, `<head>`, `<body>`)
   - Semantic HTML5 elements
   - Common attributes

2. **[Text Formatting](examples/basic/text-formatting.html)**
   - Headings (`<h1>`-`<h6>`)
   - Text emphasis (`<strong>`, `<em>`, `<mark>`)
   - Quotes and citations
   - Code and preformatted text

3. **[Lists](examples/basic/lists.html)**
   - Ordered and unordered lists
   - Description lists
   - Nested lists
   - List attributes

4. **[Images & Links](examples/basic/images-links.html)**
   - Image embedding (`<img>`)
   - Different link types
   - Image maps
   - Accessibility

5. **[Forms](examples/basic/forms.html)**
   - Form structure
   - Input types
   - Form validation
   - Accessibility

### Phase 2: CSS Fundamentals (Beginner)

Learn the basics of styling and CSS concepts:

6. **[CSS Introduction](examples/basic/css-intro.html)**
   - CSS syntax and rules
   - Ways to include CSS
   - Basic properties
   - Units and measurements

7. **[CSS Selectors](examples/basic/selectors.html)**
   - Basic selectors
   - Combination selectors
   - Attribute selectors
   - Pseudo-classes and pseudo-elements

8. **[Colors & Backgrounds](examples/basic/colors-backgrounds.html)**
   - Color formats
   - Background properties
   - Gradients
   - Transparency

9. **[Typography](examples/basic/typography.html)**
   - Font properties
   - Text styling
   - Web fonts
   - Text effects

10. **[Box Model](examples/basic/box-model.html)**
    - Margin, padding, border
    - Box sizing
    - Width and height
    - Overflow

### Phase 3: Intermediate CSS (Intermediate)

Master layout and animation techniques:

11. **[Flexbox](examples/intermediate/flexbox.html)**
    - Flex container properties
    - Flex item properties
    - Common patterns
    - Responsive design

12. **[CSS Grid](examples/intermediate/grid.html)**
    - Grid container properties
    - Grid item placement
    - Grid areas
    - Responsive grids

13. **[Positioning](examples/intermediate/positioning.html)**
    - Position values
    - Z-index and stacking
    - Common patterns
    - Best practices

14. **[Transitions](examples/intermediate/transitions.html)**
    - Transition properties
    - Timing functions
    - Interactive effects
    - Performance

15. **[Animations](examples/intermediate/animations.html)**
    - Keyframes
    - Animation properties
    - Interactive animations
    - Performance tips

### Phase 4: Advanced CSS (Advanced)

Explore modern CSS features and advanced techniques:

16. **[Responsive Design](examples/advanced/responsive-design.html)**
    - Media queries
    - Viewport meta tag
    - Responsive units
    - Mobile-first design

17. **[CSS Variables](examples/advanced/css-variables.html)**
    - Custom properties
    - Dynamic theming
    - Scopes and inheritance
    - JavaScript integration

18. **[Pseudo-elements](examples/advanced/pseudo-elements.html)**
    - ::before and ::after
    - ::first-line and ::first-letter
    - Creative applications
    - Browser support

19. **[Filters & Blends](examples/advanced/filters-blends.html)**
    - Filter functions
    - Blend modes
    - Backdrop filters
    - Visual effects

20. **[Transforms](examples/advanced/transforms.html)**
    - 2D transforms
    - 3D transforms
    - Transform origin
    - Interactive examples

21. **[Modern CSS](examples/advanced/modern-css.html)**
    - CSS Grid subgrid
    - Container queries
    - CSS nesting
    - Modern color functions
    - Cascade layers

## How to Use This Resource

### For Beginners

1. **Start from the beginning**: Follow the learning path in order
2. **Read the explanations**: Each example includes detailed explanations
3. **Experiment**: Modify the code and see what happens
4. **Practice**: Try recreating the examples from scratch
5. **Build projects**: Apply what you've learned to small projects

### For Intermediate Developers

2. **Master responsive design**: Create mobile-first layouts
3. **Learn advanced techniques**: Pseudo-elements, filters, transforms
4. **Optimize for performance**: Use hardware acceleration
5. **Contribute to the community**: Share your knowledge
