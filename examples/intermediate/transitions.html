Timing Functions

.linear { transition: all 0.5s linear; /* Constant speed */ } .ease { transition: all 0.5s ease; /* Slow start, fast middle, slow end */ } .ease-in { transition: all 0.5s ease-in; /* Slow start, fast end */ } .ease-out { transition: all 0.5s ease-out; /* Fast start, slow end */ } .ease-in-out { transition: all 0.5s ease-in-out; /* Slow start and end */ } /* Cubic bezier curves */ .custom-timing { transition: all 0.5s cubic-bezier(0.68, -0.55, 0.265, 1.55); }

Performance Considerations

/* Use transform and opacity for better performance */ .performant-transition { transition: transform 0.3s ease, opacity 0.3s ease; } /* Avoid these properties for smooth animations */ /* width, height, padding, margin, left, top */ /* Use will-change for complex animations */ .complex-animation { will-change: transform, opacity; } /* Remove will-change after animation */ .complex-animation.animation-end { will-change: auto; } /* Use hardware acceleration */ .hardware-accelerated { transform: translateZ(0); backface-visibility: hidden; }

Best Practices

/* Keep transitions short and subtle */ .good-transition { transition: all 0.2s ease; } /* Use consistent timing across your site */ :root { --transition-fast: 0.15s ease; --transition-normal: 0.3s ease; --transition-slow: 0.5s ease; } /* Consider accessibility */ @media (prefers-reduced-motion: reduce) { * { transition: none !important; } } /* Add transitions to interactive elements only */ button, a, input, .interactive { transition: var(--transition-normal); } /* Test on mobile devices */ .mobile-friendly { transition: transform 0.3s ease; /* Works well on touch */ }
