Grid vs Flexbox

/* Use Grid for: */ /* - Two-dimensional layouts (rows AND columns) */ /* - Page-level layouts */ /* - Complex grid structures */ /* - Overlapping elements */ /* - Precise control over placement */ /* Use Flexbox for: */ /* - One-dimensional layouts (row OR column) */ /* - Component-level layouts */ /* - Content alignment and distribution */ /* - Navigation bars, cards, forms */ /* They can work together! */ .grid-page { display: grid; grid-template-areas: "header" "main" "footer"; } .flex-card { display: flex; flex-direction: column; }

Advanced Grid Techniques

/* Subgrid (CSS Grid Level 2) */ .subgrid-container { display: grid; grid-template-columns: 1fr 1fr 1fr; } .subgrid-item { grid-column: span 2; display: grid; grid-template-columns: subgrid; } /* Named grid lines */ .named-lines { display: grid; grid-template-columns: [start] 1fr [middle] 2fr [end]; } .named-lines-item { grid-column: start / middle; } /* Dynamic grid with CSS variables */ :root { --columns: 3; --gap: 1rem; } .dynamic-grid { display: grid; grid-template-columns: repeat(var(--columns), 1fr); gap: var(--gap); }

Best Practices

/* Use semantic HTML with grid areas */ .page-layout { display: grid; grid-template-areas: "header header header" "nav main aside" "footer footer footer"; } /* Make grids responsive */ .responsive-grid { display: grid; grid-template-columns: repeat(auto-fit, minmax(250px, 1fr)); } /* Use minmax for flexible sizing */ .flexible-grid { grid-template-columns: minmax(200px, 1fr) minmax(300px, 2fr); } /* Consider accessibility with reordering */ .reordered-item { order: -1; /* Be careful with visual reordering */ } /* Test with different content sizes */ .test-grid { display: grid; grid-template-columns: 1fr 1fr; min-height: 0; /* Prevents overflow issues */ }
