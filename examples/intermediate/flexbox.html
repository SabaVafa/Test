Flex Item Properties

Common Flexbox Patterns

/* Center content both horizontally and vertically */ .center-content { display: flex; justify-content: center; align-items: center; } /* Holy Grail Layout */ .holy-grail { display: flex; flex-direction: column; min-height: 100vh; } .holy-grail-header, .holy-grail-footer { flex: 0 0 auto; } .holy-grail-body { flex: 1; display: flex; } .holy-grail-content { flex: 1; } /* Navigation bar */ .navbar { display: flex; justify-content: space-between; align-items: center; } /* Card layout */ .card-container { display: flex; flex-wrap: wrap; gap: 1rem; } .card { flex: 1 1 300px; }

Flexbox vs Grid

/* Use Flexbox for: */ /* - One-dimensional layouts (row or column) */ /* - Component-level layouts */ /* - Content alignment and distribution */ /* - Navigation bars, cards, forms */ /* Use Grid for: */ /* - Two-dimensional layouts (rows and columns) */ /* - Page-level layouts */ /* - Complex grid structures */ /* - Overlapping elements */ /* They can work together! */ .grid-container { display: grid; grid-template-columns: 1fr 1fr 1fr; } .flex-card { display: flex; flex-direction: column; }

Best Practices

/* Use flexbox for component layouts */ .component { display: flex; gap: 1rem; /* Modern way to add spacing */ } /* Use min-width: 0 to prevent overflow */ .flex-item { min-width: 0; } /* Use flex shorthand for better performance */ .flex-item { flex: 1; /* Instead of grow, shrink, basis separately */ } /* Consider accessibility */ .reordered-item { order: 2; } /* Test with different content sizes */ .test-container { display: flex; min-height: 0; /* Prevents overflow issues */ }
