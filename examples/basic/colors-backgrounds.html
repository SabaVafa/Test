Background Shorthand

/* Long form */ .bg-long { background-color: #ffffff; background-image: url('image.jpg'); background-repeat: no-repeat; background-position: center center; background-size: cover; background-attachment: fixed; } /* Shorthand form */ .bg-short { background: #ffffff url('image.jpg') no-repeat center center/cover fixed; } /* Gradient shorthand */ .bg-gradient-short { background: linear-gradient(45deg, #667eea 0%, #764ba2 100%); }

Color Keywords and System Colors

/* CSS Color Keywords */ .keyword-demo { color: currentColor; /* Inherits from color property */ background-color: transparent; /* Transparent background */ } /* System Colors */ .system-demo { background-color: Window; color: WindowText; border-color: ButtonFace; }

Best Practices

/* Use semantic colors */ :root { --primary-color: #007bff; --secondary-color: #6c757d; --success-color: #28a745; --warning-color: #ffc107; --danger-color: #dc3545; } /* Ensure sufficient contrast for accessibility */ .text-on-light { color: #333; /* Dark text on light background */ } .text-on-dark { color: #fff; /* Light text on dark background */ } /* Provide fallback colors */ .gradient-fallback { background-color: #667eea; /* Fallback */ background: linear-gradient(45deg, #667eea 0%, #764ba2 100%); }
