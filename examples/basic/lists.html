HTML Lists

This page demonstrates various types of HTML lists and their usage.

Unordered List (Bullet Points)

<ul> <li>First item</li> <li>Second item</li> <li>Third item</li> </ul>
  • First item
  • Second item
  • Third item

Ordered List (Numbered)

<ol> <li>First step</li> <li>Second step</li> <li>Third step</li> </ol>
  1. First step
  2. Second step
  3. Third step

Nested Lists

<ul> <li>Fruits <ul> <li>Apple</li> <li>Banana</li> <li>Orange</li> </ul> </li> <li>Vegetables <ul> <li>Carrot</li> <li>Broccoli</li> </ul> </li> </ul>
  • Fruits
    • Apple
    • Banana
    • Orange
  • Vegetables
    • Carrot
    • Broccoli
