CSS Introduction

This page demonstrates the basics of CSS (Cascading Style Sheets) and different ways to apply styles to HTML elements.

What is CSS?

CSS stands for Cascading Style Sheets. It's used to style and layout web pages, including colors, fonts, spacing, and positioning.

/* CSS Rule Structure */ selector { property: value; another-property: another-value; }

Inline CSS (Style Attribute)

<p style="color: red; font-size: 18px; background: yellow;"> This paragraph has inline styles. </p>

This paragraph has inline styles.

Pros: Quick for testing, high specificity
Cons: Hard to maintain, not reusable

Internal CSS (Style Tag)

<style> .internal-example { color: blue; font-size: 16px; background: #e6f3ff; padding: 15px; border: 2px solid #99ccff; border-radius: 5px; } </style> <p class="internal-example"> This paragraph uses internal CSS. </p>

This paragraph uses internal CSS.

Pros: Good for single pages, reusable within page
Cons: Can't be shared between pages

External CSS (Linked Stylesheet)

/* In styles.css file */ .external-example { color: green; font-size: 16px; background: #e6ffe6; padding: 15px; border: 2px solid #99ff99; border-radius: 5px; } /* In HTML file */ <link rel="stylesheet" href="styles.css"> <p class="external-example"> This paragraph uses external CSS. </p>

This paragraph uses external CSS.

Pros: Best practice, reusable across pages, easy maintenance
Cons: Requires separate file, HTTP request

CSS Comments

/* This is a CSS comment */ /* Comments can span multiple lines and are ignored by the browser */ p { color: black; /* Set text color to black */ font-size: 14px; /* Set font size */ }

Basic CSS Properties

.properties-demo { /* Text properties */ color: #333; font-family: Arial, sans-serif; font-size: 16px; font-weight: bold; text-align: center; text-decoration: underline; /* Background properties */ background-color: #f0f0f0; background-image: url('pattern.jpg'); background-repeat: no-repeat; /* Box properties */ width: 200px; height: 100px; margin: 10px; padding: 15px; border: 2px solid #ccc; border-radius: 8px; /* Display properties */ display: block; visibility: visible; opacity: 0.8; }

CSS Units

.units-demo { /* Absolute units */ width: 100px; /* pixels */ height: 2cm; /* centimeters */ margin: 1in; /* inches */ /* Relative units */ font-size: 1.2em; /* relative to parent font size */ width: 50%; /* percentage of parent */ padding: 2rem; /* relative to root font size */ margin: 10vw; /* 10% of viewport width */ height: 5vh; /* 5% of viewport height */ }

CSS Shorthand Properties

/* Long form */ .long-form { margin-top: 10px; margin-right: 20px; margin-bottom: 10px; margin-left: 20px; } /* Shorthand form */ .shorthand { margin: 10px 20px; /* top/bottom left/right */ padding: 5px 10px 15px 20px; /* top right bottom left */ border: 2px solid red; /* width style color */ background: #fff url('bg.jpg') no-repeat center; font: italic bold 16px Arial; }

CSS Inheritance

body { font-family: Arial, sans-serif; color: #333; } .container { font-size: 16px; } /* Child elements inherit font-family and color from body */ /* Child elements inherit font-size from .container */

This text inherits font-family and color from body, and font-size from container.

This span also inherits the same styles.

CSS Specificity

/* Low specificity - element selector */ p { color: black; } /* Medium specificity - class selector */ .highlight { color: blue; } /* High specificity - ID selector */ #special { color: red; } /* Very high specificity - inline style */ /* style="color: green;" */ /* Highest specificity - !important */ p { color: purple !important; }

CSS Reset vs Normalize

/* Simple CSS Reset */ * { margin: 0; padding: 0; box-sizing: border-box; } /* Normalize.css makes browsers render all elements */ /* more consistently and in line with modern standards */
