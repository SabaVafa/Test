CSS Box Model

This page demonstrates the CSS box model and how margin, padding, border, and content work together.

Box Model Overview

/* The CSS Box Model consists of: */ /* 1. Content - The actual content */ /* 2. Padding - Space around the content */ /* 3. Border - Goes around the padding */ /* 4. Margin - Space outside the border */ .box { width: 200px; /* Content width */ height: 100px; /* Content height */ padding: 20px; /* Space around content */ border: 5px solid blue; /* Border around padding */ margin: 10px; /* Space outside border */ }

Red dashed = Margin | Green solid = Padding | Orange solid = Border | Blue solid = Content

Box Sizing

/* Default box-sizing: content-box */ .content-box { box-sizing: content-box; width: 200px; padding: 20px; border: 5px solid blue; /* Total width = 200 + 40 + 10 = 250px */ } /* Border-box includes padding and border in width */ .border-box { box-sizing: border-box; width: 200px; padding: 20px; border: 5px solid blue; /* Total width = 200px (content adjusts) */ } /* Global border-box for easier layout */ *, *::before, *::after { box-sizing: border-box; }
Content-box: 200px + 40px padding + 10px border = 250px total
Border-box: Total width = 200px (content adjusts)

Display Property and Box Model

/* Block elements (full width, stack vertically) */ .block { display: block; width: 200px; margin: 10px 0; } /* Inline elements (only take needed space, flow horizontally) */ .inline { display: inline; margin: 10px; /* width/height don't work on inline elements */ } /* Inline-block (flow horizontally but accept box model properties) */ .inline-block { display: inline-block; width: 100px; height: 50px; margin: 10px; }

Overflow

.overflow-visible { overflow: visible; /* Default */ } .overflow-hidden { overflow: hidden; } .overflow-scroll { overflow: scroll; } .overflow-auto { overflow: auto; } /* Individual axes */ .overflow-xy { overflow-x: auto; overflow-y: hidden; }
Visible overflow: This content is longer than the container and will spill out
Hidden overflow: This content is longer than the container but is clipped
Scroll overflow: This content is longer than the container and shows scrollbars
Auto overflow: This content is longer than the container and shows scrollbars when needed

Best Practices

/* Use border-box for easier calculations */ *, *::before, *::after { box-sizing: border-box; } /* Remove default margins for consistent layout */ body { margin: 0; } /* Use consistent spacing */ :root { --spacing-xs: 4px; --spacing-sm: 8px; --spacing-md: 16px; --spacing-lg: 24px; --spacing-xl: 32px; } /* Handle margin collapse */ .container { padding: 1rem; } /* Use overflow for content that might exceed container */ .responsive-container { overflow: auto; max-width: 100%; }
