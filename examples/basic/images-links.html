HTML Images & Links

This page demonstrates how to use images and create links in HTML.

Basic Image

<img src="image.jpg" alt="Description of image">
200x150

Image with alt text for accessibility

Image with Size Attributes

<img src="image.jpg" alt="Description" width="300" height="200">
300x200

Image with explicit width and height

Responsive Image

<img src="image.jpg" alt="Description" style="max-width: 100%; height: auto;">
Responsive

Image that scales with viewport

Basic Link

<a href="https://www.example.com">Visit Example.com</a>
Visit Example.com

Link that Opens in New Tab

<a href="https://www.example.com" target="_blank" rel="noopener noreferrer"> Open in New Tab </a>
Open in New Tab

Email Link

<a href="mailto:email@example.com">Send Email</a>
Send Email

Phone Link

<a href="tel:+1234567890">Call +1 234-567-890</a>
Call +1 234-567-890

Image as Link

LOGO

Anchor Links (Page Navigation)
