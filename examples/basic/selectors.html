CSS Selectors

This page demonstrates different types of CSS selectors and how to target HTML elements.

Basic Selectors

/* Element selector */ p { color: red; font-weight: bold; } /* Class selector */ .highlight { color: blue; font-style: italic; } /* ID selector */ #header { color: green; text-decoration: underline; } /* Universal selector */ * { margin: 0; padding: 0; }

/* Descendant selector (space) */ div p { background: #ffe6e6; } /* Child selector (>) */ ul > li { background: #e6f3ff; } /* Adjacent sibling selector (+) */ h2 + p { background: #e6ffe6; } /* General sibling selector (~) */ h2 ~ p { background: #fff0e6; }

Attribute Selectors

/* Attribute exists */ [disabled] { opacity: 0.5; } /* Attribute equals value */ input[type="text"] { border: 2px solid red; } /* Attribute contains word */ [class~="highlight"] { background: yellow; } /* Attribute starts with */ [href^="https"] { color: green; } /* Attribute ends with */ [src$=".jpg"] { border: 1px solid blue; } /* Attribute contains substring */ [class*="demo"] { font-weight: bold; }




Pseudo-class Selectors

/* Dynamic pseudo-classes */ a:hover { color: red; } input:focus { outline: 2px solid blue; } a:active { color: green; } /* Structural pseudo-classes */ li:first-child { background: yellow; } li:last-child { background: lightblue; } li:nth-child(2) { background: lightgreen; } li:nth-child(odd) { font-weight: bold; } li:nth-child(even) { font-style: italic; } /* UI state pseudo-classes */ input:checked { margin-left: 20px; } input:disabled { opacity: 0.5; } input:enabled { background: white; } /* Language pseudo-class */ p:lang(en) { quotes: '"' '"'; }

Pseudo-element Selectors

/* ::before and ::after */ .quote::before { content: '"'; color: red; } .quote::after { content: '"'; color: red; } /* ::first-line and ::first-letter */ p::first-line { color: blue; } p::first-letter { font-size: 2em; color: red; } /* ::selection */ ::selection { background: yellow; color: black; }

Advanced Selectors

/* :not() pseudo-class */ p:not(.highlight) { color: gray; } /* :empty pseudo-class */ div:empty { background: lightgray; height: 20px; } /* :nth-of-type() */ p:nth-of-type(2) { border: 2px solid red; } /* :only-child */ span:only-child { background: yellow; } /* :root */ :root { --main-color: blue; }

Selector Specificity Examples

/* Specificity hierarchy (highest to lowest): */ /* 1. Inline styles */ /* 2. ID selectors */ #id { color: red; } /* Specificity: 100 */ /* 3. Class selectors, attribute selectors, pseudo-classes */ .class { color: blue; } /* Specificity: 10 */ [type="text"] { color: green; } /* Specificity: 10 */ :hover { color: orange; } /* Specificity: 10 */ /* 4. Element selectors, pseudo-elements */ p { color: black; } /* Specificity: 1 */ ::before { content: ""; } /* Specificity: 1 */ /* 5. Universal selector */ * { margin: 0; } /* Specificity: 0 */ /* Combining selectors adds specificity */ #id .class p { color: purple; } /* Specificity: 100 + 10 + 1 = 111 */

Best Practices

/* Use class selectors for reusable styles */ .btn-primary { background: blue; } .btn-secondary { background: gray; } /* Use ID selectors sparingly (only for unique elements) */ #main-header { position: fixed; } /* Avoid overly specific selectors */ /* Bad: */ div.container > ul.nav > li.item > a.link { } /* Good: */ .nav-link { } /* Use semantic HTML with appropriate selectors */ article { margin: 20px; } aside { float: right; }
