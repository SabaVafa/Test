Responsive Design

This page demonstrates responsive design techniques for creating websites that work across all devices.

Viewport Meta Tag

<meta name="viewport" content="width=device-width, initial-scale=1.0"> /* Common viewport settings */ <meta name="viewport" content="width=device-width, initial-scale=1.0, maximum-scale=1.0, user-scalable=no"> /* Viewport options */ /* width: device-width | specific width */ /* initial-scale: 1.0 | 0.5 | 2.0 */ /* maximum-scale: 1.0 | 2.0 */ /* minimum-scale: 1.0 | 0.5 */ /* user-scalable: yes | no */

Viewport Meta Tag: Essential for responsive design. Tells the browser how to control the page's dimensions and scaling.

width=device-width: Sets the viewport width to match the device's screen width.

initial-scale=1.0: Sets the initial zoom level when the page is first loaded.

Media Queries

/* Mobile-first approach */ .element { width: 100%; font-size: 14px; } /* Tablet */ @media (min-width: 768px) { .element { width: 50%; font-size: 16px; } } /* Desktop */ @media (min-width: 1024px) { .element { width: 33.333%; font-size: 18px; } } /* Desktop-first approach */ .element { width: 33.333%; font-size: 18px; } @media (max-width: 1023px) { .element { width: 50%; font-size: 16px; } } @media (max-width: 767px) { .element { width: 100%; font-size: 14px; } }

.responsive-grid { display: grid; gap: 20px; grid-template-columns: 1fr; /* Mobile: 1 column */ } @media (min-width: 768px) { .responsive-grid { grid-template-columns: repeat(2, 1fr); /* Tablet: 2 columns */ } } @media (min-width: 1024px) { .responsive-grid { grid-template-columns: repeat(3, 1fr); /* Desktop: 3 columns */ } } @media (min-width: 1200px) { .responsive-grid { grid-template-columns: repeat(4, 1fr); /* Large desktop: 4 columns */ } } /* Auto-fit responsive grid */ .auto-fit-grid { display: grid; grid-template-columns: repeat(auto-fit, minmax(250px, 1fr)); gap: 20px; } /* Auto-fill responsive grid */ .auto-fill-grid { display: grid; grid-template-columns: repeat(auto-fill, minmax(200px, 1fr)); gap: 20px; }

Responsive Typography

/* Fluid typography with clamp() */ .fluid-text { font-size: clamp(1rem, 2.5vw, 1.5rem); line-height: 1.6; } /* Responsive font sizes */ .responsive-text { font-size: 14px; /* Mobile */ } @media (min-width: 768px) { .responsive-text { font-size: 16px; /* Tablet */ } } @media (min-width: 1024px) { .responsive-text { font-size: 18px; /* Desktop */ } } /* Responsive line height */ .responsive-line-height { line-height: 1.4; /* Mobile */ } @media (min-width: 768px) { .responsive-line-height { line-height: 1.6; /* Tablet+ */ } } /* Viewport units for typography */ .viewport-text { font-size: 4vw; /* 4% of viewport width */ } /* Calc for typography */ .calc-text { font-size: calc(14px + 1vw); }

Best Practices

/* Mobile-first approach */ .element { /* Base styles for mobile */ width: 100%; padding: 10px; } @media (min-width: 768px) { .element { /* Enhancements for larger screens */ width: 50%; padding: 20px; } } /* Use relative units */ .responsive-element { width: 100%; max-width: 1200px; margin: 0 auto; padding: 0 1rem; } /* Optimize images */ img { max-width: 100%; height: auto; loading: lazy; } /* Touch-friendly targets */ button, .click-target { min-height: 44px; min-width: 44px; } /* Performance considerations */ .performant-animation { transform: translateX(100px); /* Better than left */ will-change: transform; /* Use sparingly */ }
