Cascade Layers

/* Define layers in order of precedence */ @layer base, components, utilities; @layer base { body { font-family: Arial, sans-serif; line-height: 1.6; } } @layer components { .button { background: #2196f3; color: white; } } @layer utilities { .text-center { text-align: center; } } /* Unlayered styles have highest precedence */ .button { background: #ff9800; /* Wins over layer */ }

CSS Shapes

This text flows around the circular shape using CSS Shapes. The shape-outside property allows text to wrap around non-rectangular shapes, creating more dynamic and interesting layouts.

Feature Detection

@supports (display: grid) { /* Grid layout supported */ .layout { display: grid; } } @supports not (display: grid) { /* Fallback for no grid support */ .layout { display: flex; } } @supports (backdrop-filter: blur(10px)) { /* Backdrop filter supported */ .glass-effect { backdrop-filter: blur(10px); } } @supports (container-type: inline-size) { /* Container queries supported */ }

Best Practices

/* Use feature detection for progressive enhancement */ @supports (display: grid) { .modern-layout { display: grid; } } /* Provide fallbacks for older browsers */ .legacy-layout { display: flex; /* Fallback */ } /* Use cascade layers for better specificity control */ @layer base, components, utilities; /* Use logical properties for internationalization */ .i18n-friendly { margin-inline: 20px; text-align: start; } /* Use modern color functions for better theming */ .themed { background: color-mix(in srgb, var(--primary) 80%, var(--secondary) 20%); } /* Use clamp() for responsive design without media queries */ .responsive { font-size: clamp(1rem, 2.5vw, 1.5rem); } /* Test in multiple browsers */ /* Use progressive enhancement approach */
