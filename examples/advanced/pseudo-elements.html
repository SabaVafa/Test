CSS Pseudo-elements

This page demonstrates CSS pseudo-elements for styling specific parts of elements and adding decorative content.

::before and ::after

.element::before { content: "Before content"; color: blue; font-weight: bold; } .element::after { content: "After content"; color: red; font-weight: bold; } /* Different content types */ .string-content::before { content: "Text"; } .image-content::before { content: url('icon.png'); } .counter-content::before { content: counter(item); } .attr-content::before { content: attr(data-text); } /* No content (for positioning) */ .empty-content::before { content: ""; display: block; width: 10px; height: 10px; }
This element has before and after pseudo-elements

::first-line

.paragraph::first-line { color: blue; font-weight: bold; font-size: 18px; text-transform: uppercase; } /* Limited properties available */ /* font properties, color, background, word-spacing, */ /* letter-spacing, text-decoration, text-transform, */ /* line-height, text-shadow, clear */
This is the first line of text that will be styled differently. The rest of the paragraph will have normal styling. Lorem ipsum dolor sit amet, consectetur adipiscing elit. Sed do eiusmod tempor incididunt ut labore et dolore magna aliqua.

::first-letter

.paragraph::first-letter { color: green; font-size: 48px; font-weight: bold; float: left; line-height: 1; margin-right: 8px; } /* Works only with block-level elements */ /* Limited to: font properties, color, background, */ /* margin, padding, border, text-decoration, */ /* text-transform, line-height, float, clear */
his is a paragraph with a styled first letter. This effect is commonly used in typography to create drop caps. The first letter is larger and styled differently from the rest of the text.

::selection

::selection { background: yellow; color: black; } /* Firefox prefix */ ::-moz-selection { background: yellow; color: black; } /* Element-specific selection */ .highlight-text::selection { background: #ffeb3b; color: #333; } /* Limited properties: color, background, cursor, */ /* outline, text-decoration, text-shadow, */ /* -webkit-text-stroke-color, -webkit-text-fill-color */
Try selecting this text to see the custom selection styling. The background and text color will change when you select it. This is useful for creating branded selection styles.

Creative Examples

/* Custom quote marks */ .quote::before { content: """; position: absolute; left: 10px; top: 5px; font-size: 48px; color: blue; } /* Custom list bullets */ .custom-list li::before { content: "»"; color: blue; font-weight: bold; margin-right: 8px; } /* Button overlay effect */ .button::before { content: ""; position: absolute; top: 0; left: -100%; width: 100%; height: 100%; background: rgba(255,255,255,0.2); transition: left 0.3s ease; } .button:hover::before { left: 0; }
This is a quote with custom pseudo-element styling. The before and after pseudo-elements create quotation marks around the content.

Advanced Techniques

/* Loading spinner with pseudo-elements */ .spinner::before, .spinner::after { content: ""; position: absolute; border: 2px solid transparent; border-radius: 50%; animation: spin 1s linear infinite; } .spinner::before { width: 40px; height: 40px; border-top-color: blue; } .spinner::after { width: 24px; height: 24px; top: 8px; left: 8px; border-top-color: orange; animation-direction: reverse; } /* Gradient text with shadow */ .gradient-text::before { content: attr(data-text); position: absolute; z-index: -1; opacity: 0.3; transform: translateX(2px) translateY(2px); } /* Image overlay */ .image-overlay::before { content: ""; position: absolute; top: 0; left: 0; right: 0; bottom: 0; background: rgba(0,0,0,0.3); opacity: 0; transition: opacity 0.3s ease; } .image-overlay:hover::before { opacity: 1; }

Interactive Elements

/* Tab indicator */ .tab::before { content: ""; position: absolute; bottom: 0; left: 0; right: 0; height: 2px; background: blue; transform: scaleX(0); transition: transform 0.3s ease; } .tab:hover::before, .tab.active::before { transform: scaleX(1); } /* Tooltip */ .tooltip::before { content: attr(data-tooltip); position: absolute; bottom: 100%; left: 50%; transform: translateX(-50%); background: #333; color: white; padding: 8px 12px; border-radius: 4px; opacity: 0; transition: opacity 0.3s ease; } .tooltip:hover::before { opacity: 1; }

Browser Support

/* ::before and ::after - IE8+ */ /* Use single colon for older browsers */ .element:before { /* IE8 */ } .element::before { /* Modern */ } /* ::first-line and ::first-letter - IE5.5+ */ /* ::selection - IE9+ (with prefix) */ ::-moz-selection { /* Firefox */ } /* ::placeholder - Modern browsers */ /* Requires vendor prefixes for older browsers */ /* ::marker - Modern browsers */ /* Firefox 68+, Chrome 86+, Safari 14+ */ /* New pseudo-elements (limited support) */ ::part() { /* Chrome 65+, Edge 79+ */ } ::slotted() { /* Chrome 65+, Edge 79+ */

Best Practices

/* Use pseudo-elements for decorative content */ .decorative::before { content: ""; /* Add decorative elements */ } /* Don't use pseudo-elements for important content */ /* Screen readers can't access pseudo-element content */ /* Use semantic HTML when possible */ <blockquote> /* Better than using ::before/::after for quotes */ /* Consider performance */ .element::before { content: ""; position: absolute; will-change: transform; /* For animated pseudo-elements */ } /* Test in different browsers */ /* Some pseudo-elements have limited support */ /* Use prefixes when needed */ .element { -webkit-text-stroke: 1px black; text-stroke: 1px black; }
